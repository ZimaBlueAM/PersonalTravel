# JR 基地旅行路线

给两个人在 iPhone 上看的北海道到东京路线大纲。

当前交互：

1. 初始只显示 6 个基地，全部未展开。
2. 单独显示 6 段基地间公共交通，默认折叠，点开看步骤、风险和来源。
3. 点加号展开下一级：基地 -> 城市 -> 可选子目的地。
4. 点任意名字打开弹窗，看图片、玩法、时间预算、交通要点、天气/体力取舍、交通审计和来源。
5. 递进关系里的交通枢纽城市用 `◎` 标出。
6. 每个弹窗都带店铺/餐厅方向和小红书、Google Maps、Tabelog/网页、近期游记评测索引。
7. 大纲顶部可按 `松弛 / 边缘 / 探店 / 交通` 临时筛选，筛选只改变视图，不改变路线骨架。

交通审计重点：

- 函馆到本州必须走 `函馆 -> 新函馆北斗 -> 北海道新干线 -> 新青森`，不是函馆站直达新青森。
- 富良野到新得区间已于 2024-04-01 铁路废止，旭川/富良野到带广不能按 JR 直通支线处理。
- 旭川到钏路、钏路到函馆均为长距离移动日。JR 可拼接，但不适合叠加远支线。
- 新增候选城市只作为替代分支：登别/洞爷湖替代札幌段重支线，层云峡替代旭川花田延展，川汤/摩周替代钏路重海岸线，大沼替代函馆城市打卡，青森/弘前用于拆跨海日，山寺/作并/秋保替代仙台第二天，横滨/镰仓/日光替代东京远郊收尾。
- 个人偏好优先级：松弛、世界边缘、探店。一个可选点如果三者都不命中，就应该删掉。
